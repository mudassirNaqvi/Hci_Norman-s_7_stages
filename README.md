
# 🧠 Norman’s 7 Stages of Action (Simple Demo)

## 📌 Project Description

This is a simple web-based demonstration of **Norman’s 7 Stages of Action** from Human-Computer Interaction (HCI).

The program shows each stage step-by-step using a real-life example of turning on a light.

---

## 🎮 How It Works

1. Click the **Start** button
2. The system will display each stage one by one:

   * Goal
   * Intention
   * Action
   * Execution
   * Perception
   * Interpretation
   * Evaluation

Each stage appears automatically with a short delay.

## 🧠 HCI Concept

This project explains **Norman’s Execution–Evaluation Cycle**:

* **Execution Stage**

  1. Goal
  2. Intention
  3. Action
  4. Execution

* **Evaluation Stage**
  5. Perception
  6. Interpretation
  7. Evaluation
